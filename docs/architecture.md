# Architecture

Fractal Intelligence Lab is organized around a geometry-first processing loop.

## Main layers
- Transducer: maps inputs into complex-plane signals.
- Fractal substrate: applies recursive kernels.
- Fbits: maintain local state and caches.
- Homeostasis: regulates stability.
- Concept network: groups recurrent attractors.
- Oneiric engine: replays and consolidates states offline.

## Design principle
Each layer should be independently testable and replaceable.
