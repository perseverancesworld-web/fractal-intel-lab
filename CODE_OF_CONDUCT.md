# Code of Conduct

## Our pledge
We are committed to making participation in Fractal Intelligence Lab a harassment-free experience for everyone.

## Expected behavior
- Be respectful.
- Be constructive.
- Critique ideas, not people.
- Give credit where due.
- Stay evidence-oriented.

## Unacceptable behavior
- Harassment, discrimination, or intimidation.
- Deliberate misinformation.
- Personal attacks.
- Sabotage of discussion or contributions.

## Enforcement
Project maintainers may remove content or restrict participation for behavior that violates this code of conduct.

## Scope
This code applies to repository discussions, issues, pull requests, and project-related communication.
